# テスト実施用データガイド

テストを実施するために必要な、初期アカウントおよび登録済み書籍の情報をまとめました。

## 1. テスト用アカウント
システムには以下のユーザーが事前登録されています。

| ユーザー名 (ID) | パスワード | ロール | 主な用途 |
|---|---|---|---|
| **admin** | adminpass | 管理者 | 書籍の登録・編集・削除のテスト |
| **user01** | password | 一般ユーザー | 貸出・返却・検索の通常テスト |
| **user02** | password | 一般ユーザー | 他人による貸出データの確認用 |

## 3. データの初期化方法
テスト実施中にデータを汚してしまい、最初からやり直したい場合は、以下のコマンドを実行してください。
（すべての追加された貸出履歴や変更内容はリセットされます）

```bash
python database.py init
```

※これにより、上記の一覧通りの状態に戻ります。
